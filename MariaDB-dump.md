MariaDB dump export instructions

### Installation

1. Go to localhost -> Databases -> casedb/(whatever the name be for you )-> Tools -> Dump database

2. Then make sure all tables are checked and database case db ,press next

3. Then click on start and the file will be saved in Output folder
